El ORM en Djnago es un sistema para relacionar modelos o base de datos con objetos de Python, Django crea automaticamente un script al crear una aplicacion, dentro de este script se importa automaticamente un objeto proveniente de Django.db llamado models, para utilizar el ORM de Django se deben instanciar los objetos o tablas a crear directamente del objeto importado.
El objeto a crear corresponde a una tabla dentro de la basa de datos, y cada atributo del objeto correspondiente a una columna de su tabla en su base de datos. el ORM de Django toma este objeto y lo convierte a consultas SQL para poder crear, leer, actualizar, y eliminar datos de una forma orientada a objetos, donde cada tabla cuenta cin atributos y metodos.
